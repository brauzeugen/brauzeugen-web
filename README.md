# Brauzeugen

The Brauzeugen web app and token management system.

## Setup

Requires:
* Ruby 2.5
* Postgres 9+

You can set the project up locally by running:

    rbenv install
    gem install bundler
    bin/setup

## Run on Docker

    docker-compose up -d
